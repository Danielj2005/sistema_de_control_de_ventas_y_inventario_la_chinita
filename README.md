## 🌎 English Version

### 📝 Description
This project is a comprehensive **Enterprise Resource Planning (ERP)** solution developed as a professional degree project for Computer Engineering. It automates inventory tracking, sales processing, and financial reporting for retail businesses.

### 🚀 Key Features
* **Inventory Management:** Real-time tracking of stock levels with automated low-stock alerts.
* **Sales Module:** Fast Point-of-Sale (POS) interface with AJAX-powered product search.
* **User Roles:** Secure authentication and authorization (Admin/Staff) using session management.
* **Reporting:** Dynamic generation of sales reports and inventory history.
* **Efficiency:** Developed using **AI-assisted coding** to optimize delivery times and ensure SQL query efficiency.

### 🛠️ Tech Stack
* **Backend:** PHP (MVC Architecture), PDO for secure database transactions.
* **Frontend:** JavaScript (jQuery/AJAX), Bootstrap 5, HTML5, CSS3.
* **Database:** MySQL (Relational Schema).
* **Workflow:** Git, AI-driven development for rapid prototyping.

## 🇪🇸 Versión en Español

### 📝 Descripción
Este proyecto es una solución integral de **Planificación de Recursos Empresariales (ERP)** desarrollada como proyecto de grado para Ingeniería en Informática. Automatiza el seguimiento de inventarios, el procesamiento de ventas y la generación de informes financieros.

### 🚀 Características Principales
* **Gestión de Inventario:** Seguimiento en tiempo real de niveles de stock con alertas automáticas.
* **Módulo de Ventas:** Interfaz de Punto de Venta (POS) rápida con búsqueda de productos mediante AJAX.
* **Roles de Usuario:** Autenticación y autorización segura (Admin/Empleado) mediante manejo de sesiones.
* **Reportes:** Generación dinámica de reportes de ventas e historial de movimientos.
* **Eficiencia:** Desarrollado utilizando **IA asistida** para optimizar los tiempos de entrega y garantizar la eficiencia de las consultas SQL.

### 🛠️ Stack Tecnológico
* **Backend:** PHP (Arquitectura MVC), PDO para transacciones seguras.
* **Frontend:** JavaScript (jQuery/AJAX), Bootstrap 5, HTML5, CSS3.
* **Base de Datos:** MySQL.
* **Workflow:** Git, desarrollo impulsado por IA para prototipado rápido.
